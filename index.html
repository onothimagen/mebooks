How to use this list

All of the entries include a link to download a version of the text encoded using the EPUB file format for ebooks.

On most phones, tablets and computers, downloading an EPUB file will automatically open in the default Ebook reader on the device. If this doesn't happen automatically, save the file in your Downloads folder, and open it using your Ebook reader app.
